# Proyecto Juego "Escapa del Laberinto"

## Descripción
Este proyecto consiste en la creación de un juego 2D en Python que simula un laberinto. El jugador y los enemigos se mueven en tiempo real dentro de este laberinto, el cual se genera aleatoriamente al inicio de cada partida. El jugador tiene que escapar de los cazadores (Modo Escapa) o ser el cazador que persigue a otros jugadores (Modo Cazador).

### Modos de Juego:
- **Modo Escapa**: El jugador debe huir de los cazadores. Si es alcanzado por uno, pierde la partida. Si llega a la salida antes de ser atrapado, gana. El puntaje depende del tiempo que tarda en escapar.
- **Modo Cazador**: El jugador se convierte en el cazador y su objetivo es atrapar a otros cazadores. Los cazadores atrapados reaparecen después de un tiempo.

## Requisitos
- **Python 3.x**  
- **Tkinter**: Para la interfaz gráfica (viene con Python).
- **Pygame**: Para los efectos de sonido (instalar con `pip install pygame`).
